## 🚀 Featured Project — GestureCTRL (Hands‑Free Desktop Control)
Webcam-based, hands-free **desktop control** app: you **train hand gestures** and map them to OS actions (mouse + media/system shortcuts + custom hotkeys).

**Repo:** https://github.com/GovindUpadhyay13/Gesture-control

### ✨ Capabilities
- 🖱️ Cursor: move, click, right-click, drag, scroll  
- 🎵 Media/system: play/pause, next/prev, volume  
- 🪟 Shortcuts: switch windows, close tabs, screenshot, lock screen  
- ⌨️ Custom actions via UI / `data/gestures.json`

### 🧩 Architecture (high level)
| Layer | Tech |
|---|---|
| UI | React + Vite |
| Service/Backend | Node.js + Express + WebSocket |
| Gesture ML | Python + MediaPipe + scikit-learn (KNN) |
| OS Control | PyAutoGUI |
